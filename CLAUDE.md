# CLAUDE.md

leptos-floating provides floating UI positioning primitives for Leptos
applications, similar to Floating UI for JavaScript.

## Development

Do NOT use `cargo` commands directly.

```bash
# Testing
just test-rust          # Unit tests

# Code quality
just format-rust true   # Format code
just lint-rust          # Lint code
just pre-commit         # Run all checks

# See all commands
just
```

## Requirements

These are installed via Flox.

- Rust 1.88+
- Leptos 0.8+
